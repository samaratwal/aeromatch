❓ QUICK TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Q: Frontend not loading?
A: Make sure you have modern browser (Chrome 60+, Firefox 55+, Safari 11+)

Q: Backend connection fails?
A: Ensure backend running: python aeromatch_backend.py
   Check port 5000: curl http://localhost:5000/api/health

Q: No search results?
A: Try with just application type (no numbers)
   Frontend database should have 25 airfoils loaded

Q: Canvas drawing not working?
A: Must use local server, not file:// protocol
   python -m http.server 8000

Q: Want to add more airfoils?
A: Edit DB array in JavaScript or use backend database
   Can support 1,600+ with API
